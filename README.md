# 보라매 뉴스레터

공군 훈련소 훈련병들을 위한 보라매 뉴스레터입니다.

## 사용 API

- [Corona-19-API](https://api.corona-19.kr/)
- [OpenWeather API](https://openweathermap.org/api)


## 파일 설정
```.env``` 파일을 ```/backend``` 디렉토리 내부에 생성한뒤 아래와 같이 적어줍니다.

```
DB_URL=DB주소
WEATHER_KEY=API키
COVID_KEY=API키
LETTER_PASSWORD=인편 비밀번호
```
